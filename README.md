#  Fuel Estimator (Research Software Engineering in Python for Reproducible Computational Science Project)

## Description
This project implements a trip fuel consumption estimator using Monte Carlo simulations. 
It utilizes the **Numba** library for high-performance calculations and the **Pint** library for physical unit validation.
This project contains automated testing and documentation deployment.
